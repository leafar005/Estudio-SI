# Portal de Estudio SI — Sistemas Inteligentes

Portal web estático de estudio interactivo para la asignatura de **Sistemas Inteligentes (SI)**. 
Permite repasar conceptos teóricos, practicar con banco de preguntas, cuestionarios interactivos y flashcards.

## 📚 Temario

### Parte Simbólica
- 🤖 1. Introducción a la IA
- 🔍 2. Búsqueda
- 🧠 3. Representación
- ⚖️ 4. Razonamiento
- 📋 5. Planificación

### Parte Subsimbólica
- 🌐 6. Sistemas Conexionistas
- ⚡ 7. Neurona Formal y RNA
- ➡️ 8. Redes Feedforward
- 🔄 9. Redes Autoorganizativas
- 🧬 10. Computación Evolutiva

## 🚀 Características

- **Quiz Interactivo**: Motor de preguntas con corrección instantánea y justificación detallada. Cuenta con simulacros completos de la parte simbólica y de la parte subsimbólica.
- **Flashcards**: Sistema de tarjetas de memoria por cada tema.
- **Estadísticas**: Historial de simulacros, precisión de preguntas y sistema de logros.
- **Banco de Preguntas**: Directorio completo y filtrable para repasar.
- **Apuntes y Guías**: Material teórico de consulta estructurado por bloques.

## 🛠️ Tecnologías

- Vanilla JS (sin frameworks)
- HTML5 / CSS3 Moderno (Glassmorphism, Bento Grid)
- localStorage para guardar progreso
- Responsive Design
